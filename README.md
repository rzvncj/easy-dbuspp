# easy-dbuspp

Modern C++ D-Bus wrapper.

## Building

```
meson setup build
cd build
meson compile
```

## Built with

* [glib-2](https://docs.gtk.org/glib/) - D-Bus implementation
* [ASIO](https://think-async.com/Asio/) - Only for `thread_pool`
